## Getting Started

```bash
1. Clone the repo
2. Create .env file in the root folder and paste below content in file
2. npm i
3. npm run dev
```

## Environment Variables

 ```
NEXT_PUBLIC_API_PATH=http://localhost:3001/
 ```

## Build

```
1. npm run build
2. npm start
```

## Lint

```
1. npm run lint
2. npm run lint:fix 
```

## JsDoc

```
npm run docs
```

Navigate to [http://localhost:3000](http://localhost:3000).
